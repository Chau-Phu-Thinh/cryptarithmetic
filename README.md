# Installation

```bash
cd to/your/path
git clone https://github.com/Chau-Phu-Thinh/cryptarithmetic?tab=readme-ov-file
```

# Compile, run and clean

```Terminal
# BUILD
make

# RUN OPTIONS
## bruteforce
make run-brute-force

## long-mul
make run-long-mul

## column-carry
make run-column-carry

# CLEAN
make clean
```
